# React 应用模板

基于 Create React App 的现代 React 应用模板，支持 TypeScript。

> [English](./README.md) | 简体中文

## 🚀 特性

- ⚛️ **React 18** - 最新的 React 特性和 Hooks
- 🔷 **TypeScript** - 完整的类型安全
- 🎨 **CSS 支持** - 内置 CSS 和 CSS 模块支持
- 🧪 **测试** - 包含 Jest 和 React Testing Library
- 📦 **构建优化** - 生产级构建，支持代码分割
- 🔧 **ESLint** - 代码质量和一致性保证

## 🏁 快速开始

### 安装

```bash
# 安装依赖
npm install
```

### 开发

```bash
# 启动开发服务器 http://localhost:3000
npm start
```

应用将自动在默认浏览器中打开。

### 构建

```bash
# 构建生产版本
npm run build
```

build 文件夹将包含优化的生产文件。
